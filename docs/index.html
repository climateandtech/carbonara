what is carbonara

Carbonara is a lightweight, open-source code editor plugin that helps developers measure and reduce the environmental impact of their software. By monitoring the energy consumption and carbon emissions of both your code and cloud infrastructure, Carbonara empowers you to make climate-conscious decisions during development.
