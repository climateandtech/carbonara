# Carbonara Built-in Analyzers

This directory contains built-in analyzers that are part of the Carbonara core.

## Structure

Built-in analyzers should be placed in this directory and follow the analyzer interface defined in the registry system.

## Available Analyzers

Built-in analyzers will be added here as separate PRs to maintain clean development workflow.

## Adding New Analyzers

1. Create your analyzer file in this directory
2. Follow the analyzer interface patterns
3. Update the tools registry to include your analyzer
4. Add appropriate tests

For examples of analyzer implementations, see the separate analyzer PRs.
